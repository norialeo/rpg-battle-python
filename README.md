# 🎮 Sistema de Batalha RPG em Python

Este projeto consiste em uma simulação de batalha em turnos desenvolvida em Python utilizando programação orientada a objetos (POO).

O sistema permite que um herói enfrente um inimigo em combate, com opções de ataque normal e ataque especial, demonstrando conceitos fundamentais de modelagem de sistemas e lógica de jogo.

---

## ⚙️ Funcionalidades

- Sistema de batalha em turnos
- Ataques normais e especiais
- Cálculo de dano com base em nível
- Sistema de vida dos personagens
- Exibição de atributos dos personagens
- Interação via terminal

---

## 🛠 Tecnologias utilizadas

- Python
- Programação Orientada a Objetos (POO)
- Biblioteca `random`

---

## 🧠 Conceitos aplicados

- Herança
- Encapsulamento
- Classes e objetos
- Métodos e atributos privados
- Lógica de controle de fluxo
